Configuration Google Sheets

Pour exporter directement vos conversations Teams vers Google Sheets, vous devez configurer un script Google Apps. Suivez ces etapes :

Etape 1 : Ouvrir votre Google Sheet

1Ouvrir la feuille de calcul
Ouvrez votre Google Sheet :
https://docs.google.com/spreadsheets/d/1PyHVc6MWJlPs61mt8CdnWLaxyth47EUk3Z9cIQj6EY0/edit

Etape 2 : Creer le script Apps Script

2Ouvrir l'editeur de scripts
Dans le menu, cliquez sur Extensions > Apps Script
3Coller le code
Supprimez le code existant et collez le code suivant : 
function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const messages = data.messages;

    if (!messages || !Array.isArray(messages)) {
      return ContentService.createTextOutput(JSON.stringify({
        success: false,
        error: 'Invalid data format'
      })).setMimeType(ContentService.MimeType.JSON);
    }

    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    sheet.clear();

    // Add headers
    sheet.getRange(1, 1, 1, 4).setValues([['Date', 'Heure', 'Auteur', 'Message']]);
    const headerRange = sheet.getRange(1, 1, 1, 4);
    headerRange.setFontWeight('bold');
    headerRange.setBackground('#667eea');
    headerRange.setFontColor('#ffffff');

    // Add data
    if (messages.length > 0) {
      const rows = messages.map(msg => [
        msg.date || '',
        msg.time || '',
        msg.author || '',
        msg.content || ''
      ]);
      sheet.getRange(2, 1, rows.length, 4).setValues(rows);
    }

    sheet.autoResizeColumns(1, 4);
    sheet.getRange(2, 4, Math.max(1, messages.length), 1).setWrap(true);
    sheet.setColumnWidth(4, 500);

    return ContentService.createTextOutput(JSON.stringify({
      success: true,
      rowsAdded: messages.length
    })).setMimeType(ContentService.MimeType.JSON);

  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({
      success: false,
      error: error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService.createTextOutput(JSON.stringify({
    status: 'ok',
    message: 'Teams Exporter API ready'
  })).setMimeType(ContentService.MimeType.JSON);
}
4Sauvegarder
Appuyez sur Ctrl+S (ou Cmd+S sur Mac) pour sauvegarder.

Etape 3 : Deployer le script

5Nouveau deploiement
Cliquez sur Deployer > Nouveau deploiement
6Configurer le deploiement
  • Type : Application Web
  • Executer en tant que : Moi
  • Qui peut acceder : Tout le monde
7Copier l'URL
Apres le deploiement, copiez l'URL fournie. Elle ressemble a :
https://script.google.com/macros/s/AKfycbx.../exec

Etape 4 : Utiliser dans l'extension

8Configurer l'extension
  1. Ouvrez Teams sur la conversation souhaitee
  2. Cliquez sur l'icone de l'extension
  3. Selectionnez Google Sheets
  4. Collez l'URL du script
  5. Cliquez sur Exporter
C'est tout ! Vos messages Teams seront maintenant exportes directement dans votre Google Sheet.
Note : Si vous modifiez le script, vous devez creer un nouveau deploiement pour que les changements soient pris en compte.
